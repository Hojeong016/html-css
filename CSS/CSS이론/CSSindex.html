CSS가 뭐지?

CSS(Cascading style sheet)는 시각적으로 꾸미는 언어입니다. HTML에 꾸미기 기능을 넣지 않았을까요?
가독성과 효율성을 위해서 입니다. 그럼 간단하게 CSS 코드를 적용시키는 방법을 알아볼까요?

[style]= CSS코드
[p]= 선택자
[{ }]= 적용범위 / 중괄호 안의 모든 코드는 선택자 태그에 적용된다.
[;]=마침 기호/세미콜론 '명령이 끝났다'

선택자 태그


  • 전체 선택자
    • head를 포함한 HTML 문서 내의 모든 요소를 선택
    • 태그 = *
  • 선택자 태그
    • 지정된 태그를 가지는 요소를 선택합니다.
    • 태그 = 태그명
  • 아이디 선택자
  • 아이디를 지정하여 일치하는 태그를 선택합니다. 아이디 중복 불가
  • 태그 = 아이디
  • 클래스 선택자
    • 클래스명과 일치하는 요소를 선택합니다. 하나 이상 지정할 수 있습니다.
    • 태그 = .
  • 복합 선택자
  • 조상 하위에 있는 자손을 선택합니다. 조상과 자손은 띄어쓰기를 통해 구분합니다.
